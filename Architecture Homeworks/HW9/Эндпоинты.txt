1) Сервис агента:
- POST /authorization/login - авторизация
- POST /tournament/connect - подать заявку на участие. Вернет 401, если пользователь не авторизован
- POST /tournament/start - начать свой турнир. Вернет 401, если пользователь не авторизован
- POST /battle/start - начинает бой. Вернет 401, если пользователь не авторизован
- POST /battle/connect - присоединиться к бою. Вернет 401, если пользователь не авторизован
- GET /start - запускает агента и устанавливает соединение с пользователем
- GET /tournament/future - список будущих турниров
- GET /tournament/past - список прошедших турниров
- GET /tournament/present - список текущих турниров

2) Сервис турниров:
- POST /connect - подать заявку на участие. Вернет 401, если пользователь не авторизован
- POST /start - начать свой турнир. Вернет 401, если пользователь не авторизован
- GET /future - список будущих турниров
- GET /past - список прошедших турниров
- GET /present - список текущих турниров

3) Сервис боев:
- POST /start - начинает бой. Вернет 401, если пользователь не авторизован
- POST /connect - присоединиться к бою. Вернет 401, если пользователь не авторизован
- GET /create - создает новый бой

4) Сервис авторизации:
- POST /login - авторизует пользователя. Получает логин и пароль

5) Сервис пользователей:
- POST /notification/show - показ пользователю уведомления
- POST /check - проверяет, существует ли такой пользователь
- POST /create - создает нового пользователя
- POST /rating - добавляет рейтинг пользователю
- GET /user - получить данные о пользователе

6) Сервис рейтинга:
- POST /user/rating - добавить рейтинг пользователю

7) Сервис уведомлений:
- POST /user/notification - уведомить пользователя
